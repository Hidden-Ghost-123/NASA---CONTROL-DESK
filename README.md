# NASA Mission Simulator

A fast-paced 2D spaceflight simulator built with **Python** and **Pygame**. Launch a rocket from Earth, manage fuel and engine failures, achieve orbit, avoid dangerous space debris and deploy satellites within precise orbital windows to complete your mission.

## Features

*  Realistic launch-to-orbit progression
*  Multiple mission difficulty modes
*  Fuel management and stage separation
*  Random engine failures and fuel leaks
*  Satellite deployment mechanics
*  Dynamic orbital debris avoidance
*  Mission scoring and high-score tracking
*  Particle effects, screen shake and animated HUD

## Controls

| Key   | Action                |
| ----- | --------------------- |
| Space | Toggle rocket engine  |
| R     | Restart failed engine |
| D     | Deploy satellite      |
| Enter | Confirm / Restart     |
| H     | View high scores      |
| Esc   | Return to menu / Quit |

## Objective

Reach orbit at **80,000 m**, successfully deploy **3 satellites**, and survive the hazards of space to achieve the highest score possible.

## Mission Modes

### STANDARD

* Balanced fuel and hazard rates
* Ideal for new players
* Time bonus ×2

### EXPRESS

* Reduced fuel supply
* Fewer failures
* Higher score multipliers

### DEBRIS STORM

* Increased orbital debris
* Higher failure rates
* Greatest reward for successful missions
